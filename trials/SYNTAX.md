# SYNTAX

**Synergy Between PCI with Taxus and Cardiac Surgery**

---

## Quick Facts

| Field | Value |
|-------|-------|
| **Year** | 2009 (1-year), 2013 (5-year) |
| **Journal** | NEJM, Lancet |
| **DOI** | 10.1056/NEJMoa0804626 |
| **ClinicalTrials.gov** | NCT00114972 |
| **Bottom Line** | CABG superior to PCI for complex CAD; created the SYNTAX score |

---

## Design

| Parameter | Details |
|-----------|---------|
| **Type** | Randomized controlled trial |
| **N** | 1,800 |
| **Centers** | 85 sites, US and Europe |
| **Population** | 3-vessel disease and/or left main CAD |
| **Mean Age** | 65 years |
| **Randomization** | 1:1 |

### Key Design Feature
- **All-comers trial** — included left main and 3VD
- **Heart Team consensus** required for randomization
- Created **SYNTAX score** — quantifies CAD complexity

### Arms

| Arm | N | Intervention |
|-----|---|--------------|
| PCI | 903 | TAXUS paclitaxel-eluting stent |
| CABG | 897 | Standard coronary bypass |

---

## The SYNTAX Score

### What It Measures
- Number of lesions
- Location (proximal vs distal, left main)
- Complexity (bifurcation, CTO, calcification, tortuosity)
- Length and angulation

### Scoring Categories
| Score | Complexity | Implication |
|-------|------------|-------------|
| 0-22 | Low | PCI and CABG similar outcomes |
| 23-32 | Intermediate | CABG trending better |
| ≥33 | High | CABG clearly superior |

---

## Results

### 1-Year Primary Endpoint (MACCE)

| Outcome | PCI | CABG | P-value |
|---------|-----|------|---------|
| **MACCE** | **17.8%** | **12.4%** | **0.002** |

**CABG superior overall**

### 1-Year Components

| Outcome | PCI | CABG |
|---------|-----|------|
| Death | 4.4% | 3.5% |
| MI | 4.8% | 3.3% |
| Stroke | 0.6% | 2.2% |
| Repeat revasc | 13.5% | 5.9% |

### 5-Year Results

| Outcome | PCI | CABG | P-value |
|---------|-----|------|---------|
| **MACCE** | **37.3%** | **26.9%** | **<0.001** |
| Death | 13.9% | 11.4% | NS |
| MI | 9.7% | 3.8% | <0.001 |
| Stroke | 2.4% | 3.7% | NS |
| Repeat revasc | 25.9% | 13.7% | <0.001 |

## Clinical Impact

### What Changed
1. **SYNTAX score became universal** — now standard in cath labs
2. **Heart Team concept formalized** — collaborative decision-making
3. **Risk stratification** — objective basis for PCI vs CABG discussions
4. **Low complexity = PCI reasonable** — not all 3VD needs surgery
5. **Guidelines shaped** — Class I CABG for high SYNTAX, Class IIb for low

### Guideline Applications

| SYNTAX Score | PCI | CABG |
|--------------|-----|------|
| 0-22 | Class IIa | Class I |
| 23-32 | Class III (harm) | Class I |
| ≥33 | Class III (harm) | Class I |

---

## SYNTAX II Score

### Evolution
- Original SYNTAX = anatomy only
- SYNTAX II adds clinical factors:
  - Age
  - Creatinine clearance
  - LVEF
  - Sex
  - Peripheral vascular disease
  - COPD

### SYNTAX II Application
- Predicts 4-year mortality for PCI vs CABG
- More individualized than anatomy alone
- Available as online calculator

---

## Limitations

1. **First-generation DES** — TAXUS stent; current DES are better
2. **No FFR use** — angiographic guidance only
3. **High repeat revasc in PCI** — 25.9% at 5 years
4. **Stroke with CABG** — 3.7% at 5 years (higher than PCI)
5. **2009 era medicine** — statins, antiplatelets have improved

## SYNTAX Legacy

### Descendant Trials

| Trial | Question | Result |
|-------|----------|--------|
| SYNTAX | PCI vs CABG overall | CABG superior overall |
| EXCEL | PCI vs CABG left main | PCI non-inferior (controversial) |
| NOBLE | PCI vs CABG left main | CABG superior |
| FAME 3 | FFR-PCI vs CABG | PCI non-inferior? (debated) |

### What SYNTAX Established
- The SYNTAX score
- The Heart Team
- Risk-stratified revascularization
- Foundation for all subsequent trials

---

## Clinical Pearls

1. **SYNTAX score should be calculated** — not eyeballed
2. **Low SYNTAX (0-22) = PCI reasonable** — especially if anatomy favorable
3. **High SYNTAX (≥33) = CABG preferred** — unless surgical risk prohibitive
4. **Intermediate = judgment call** — patient factors, preferences matter
5. **Stroke vs repeat revasc trade-off** — CABG more stroke, PCI more revasc

---

## Key Takeaway for Practice

> SYNTAX gave us the **objective language** for PCI vs CABG decisions. The SYNTAX score isn't perfect, but it standardized complexity assessment. For **low SYNTAX (0-22)**, PCI is a reasonable alternative. For **high SYNTAX (≥33)**, CABG remains superior — even with modern DES. The Heart Team conversation starts with: "What's the SYNTAX score?"
